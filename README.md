# Azure Infrastructure Lab

This project was created to practice cloud infrastructure concepts using Microsoft Azure.
The environment was built manually using Azure Portal, focusing on networking, virtual machines, security rules and resource organization.

This lab is part of my preparation for the AZ-104 (Microsoft Azure Administrator) certification.

## Resources created

The following resources were configured in this lab:

* Resource Groups
* Virtual Networks (VNET)
* Subnets
* Network Security Groups (NSG)
* Virtual Machines
* Azure SQL Database
* Public IP
* Network communication between VNets
* Basic security rules using NSG

---

## Objective

The objective of this project is to practice Azure administration skills, including:

* Virtual network configuration
* Subnet segmentation
* NSG security rules
* Virtual machine deployment
* Database service creation
* Cloud architecture organization
* Troubleshooting connectivity

This lab simulates a basic enterprise environment with separated networks and controlled access.

---

## Deployment

This environment was created manually using Azure Portal for learning purposes.

Steps performed:

1. Created Resource Groups
2. Created Virtual Networks
3. Created Subnets
4. Configured Network Security Groups
5. Deployed Virtual Machines
6. Configured SQL Database
7. Configured network communication between VNets
8. Tested connectivity between resources
9. Documented architecture using draw.io

Future versions of this project will include Infrastructure as Code (Bicep / Terraform).
